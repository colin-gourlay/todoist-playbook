# Radio Show Post Production

A publishing and archiving workflow for completing all post-broadcast tasks after a radio show.

This template covers everything that needs to happen after the live broadcast — from finalising and publishing site content through to archiving assets and capturing reflections.

---

## Objective

- Finalise and publish the Hugo site with show notes and media
- Distribute the recording via Mixcloud and social channels
- Archive all show assets securely
- Capture reflections to improve future shows

---

## When to Use

- Immediately after every broadcast
- As part of the Radio Show Week Kit bundle
- When performing a batch catch-up after missed post-production steps

---

## Structure Overview

1. Publishing
2. Media Distribution
3. Archiving
4. Reflection

---

## Import Instructions

1. Download `template.csv`
2. Create a new project in Todoist
3. Import from CSV
4. Rename the project to: `Radio Show Post Production – [date]`
5. Work through each section on broadcast day or the day after
