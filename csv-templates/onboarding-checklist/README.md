# Onboarding Checklist

A structured first-90-days checklist for starting a new job, covering admin, relationships, learning, and early wins.

---

## Objective

- Complete essential admin and access setup from day one
- Build key relationships quickly and intentionally
- Learn how the team and organisation works
- Deliver early wins and establish credibility
- Set the foundation for long-term performance

Estimated duration: 3 hours (spread across 90 days).

---

## When to Use

- When you start a new role (employment or contract)
- When you transition into a new team within the same organisation
- As a reference for managers onboarding a new hire

---

## Structure Overview

1. Day 1 — Admin & Access
2. Week 1 — People & Context
3. Week 2–4 — Learning
4. Month 1–3 — Early Wins
5. Ongoing — Relationships

---

## Import Instructions

1. Download `template.csv`
2. Create a new project in Todoist
3. Import from CSV
4. Rename project to: `Onboarding – [Company Name]`
5. Work through sections in order, adapting tasks to your specific role and organisation
