# Radio Show Core Workflow

The core weekly workflow for preparing and delivering a radio show — from creative preparation through to the live broadcast.

This template is designed to be used **every week** and covers the essential tasks required to produce and deliver the show.

---

## Objective

- Prepare the show theme, playlist, and running order
- Handle pre-broadcast logistics and travel preparation
- Set up the studio and run a confident live broadcast

Estimated duration: varies per week.

---

## When to Use

- At the start of every show week
- As the foundation template in the Radio Show Week Kit bundle
- When onboarding a new presenter or producer

---

## Structure Overview

1. Creative Preparation
2. Pre-Broadcast Logistics
3. Studio Setup
4. Live Broadcast

---

## Import Instructions

1. Download `template.csv`
2. Create a new project in Todoist
3. Import from CSV
4. Rename the project to: `Radio Show – Week of [date]`
5. Work through each section in order leading up to and including broadcast day
