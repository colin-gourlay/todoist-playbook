## 📋 Baseline Audit Checklist

Derived from the [Socials Health & Optimisation Checklist](../csv-templates/socials-health-and-optimization-checklist/) template.

### Cross-Platform Inventory

- [x] Primary public-facing profile identified: GitHub repository
- [x] No dormant or abandoned accounts linked from the repository
- [ ] Username/handle consistency confirmed across all platforms

### Profile Basics

- [x] Repository description is accurate and current
- [x] README links are active and point to correct destinations
- [x] Repository URL is correct and publicly accessible
- [ ] GitHub Pages URL is set in the About panel (if deployed)

### Discoverability

- [x] README includes a clear value proposition in the opening paragraph
- [x] README includes a Start Here section for first-time visitors
- [ ] Repository topics/tags include all relevant keywords (`todoist`, `productivity`, `task-management`, `workflow-automation`, `gtd`, `github-actions`, `templates`)
- [ ] Repository is discoverable via GitHub search for "todoist templates"

### Consistency

- [x] Canonical naming and description is documented (see table above)
- [ ] Canonical description is applied to the GitHub About panel
- [ ] Any external posts or shares use the canonical URL and description
