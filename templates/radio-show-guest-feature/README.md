# Radio Show Guest Feature

A workflow for preparing and running a guest or feature segment within a radio show.

This template is **optional** and should only be applied in weeks where the show includes an interview, artist spotlight, special feature, or festival coverage.

---

## Objective

- Confirm and research the guest or feature subject
- Prepare interview questions and talking points
- Handle promotional preparation for the guest appearance
- Capture post-interview notes and follow-up actions

---

## When to Use

- When the show includes a live or pre-recorded interview
- When featuring an artist spotlight or in-depth review
- During festival coverage or special edition shows
- When onboarding a recurring guest contributor

---

## Structure Overview

1. Guest Preparation
2. Promotion Preparation
3. Post-Interview

---

## Import Instructions

1. Download `template.csv`
2. Create a new project in Todoist (or add tasks to an existing show project)
3. Import from CSV
4. Rename the project to: `Guest Feature – [Guest Name]`
5. Work through sections in the days leading up to and following the broadcast
