# SaaS Spin-Up

A comprehensive checklist for onboarding a new Software as a Service (SaaS) subscription — covering everything from initial evaluation through to account security, billing, integrations, learning, community engagement, and ongoing maintenance.

---

## Objective

- Evaluate the tool before committing to a paid plan
- Set up the account securely from day one
- Manage billing and renewals proactively
- Integrate the tool into your existing workflows and devices
- Build a learning and community following to maximise long-term value
- Document the tool for future reference

Estimated duration: 60–120 minutes (spread across initial days).

---

## When to Use

- When starting a free trial of a new SaaS product
- When subscribing to a new paid tool
- When inheriting or taking over a team subscription

---

## Structure Overview

1. Research & Evaluation
2. Account Setup & Security
3. Billing & Cost Management
4. Integrations & Configuration
5. Install on Devices
6. Onboarding & Learning
7. Community & Social Following
8. Documentation & Knowledge Management
9. Ongoing Maintenance

---

## Security Best Practices

- Always use an anonymised email address when signing up
- Never reuse passwords — generate via a password manager
- Enable 2FA immediately on account creation
- Rotate API keys on first use and store them securely
- Review data residency and privacy settings before entering any sensitive data
