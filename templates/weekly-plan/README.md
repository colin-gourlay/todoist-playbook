# Weekly Plan

A Sunday planning ritual to start Monday with a decided week.

This template is the start-of-week half of the two-part weekly system:

- [Weekly Close](../weekly-close/) for Friday reflection and closure
- [Weekly Plan](../weekly-plan/) for Sunday planning and prioritization

---

## Objective

- Convert reflection into a clear weekly plan
- Prioritize the highest-leverage commitments
- Protect focus time on the calendar before the week starts

Estimated duration: 30 minutes.

---

## When to Use

- Sunday evening (recommended)
- Sunday afternoon if evenings are unavailable

---

## Structure Overview

1. Re-orient
2. Plan the Future
3. Set Intentions

---

## Why This Works

Weekly planning is intentionally forward-looking. It follows a planning-first pattern:

- Reconfirm what still matters after Friday close
- Decide priorities before meetings and reactive work expand
- Place deep-work blocks first, then everything else around them

---

## Import Instructions

1. Download `template.csv`
2. Create a new project in Todoist
3. Import from CSV
4. Rename project to: `Weekly Plan - [DD/MM]`
5. Complete on Sunday evening before the workweek starts
