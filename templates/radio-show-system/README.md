# Radio Show System

End-to-end workflow for preparing, delivering and publishing a weekly radio show.

This template encodes the full lifecycle:
- Creative preparation
- Studio logistics
- Broadcast delivery
- Publishing
- Promotion
- Reflection

---

## Recommended Usage

1. Duplicate as a new project named:
   `Sundown Sessions – Week of DD/MM`
2. Schedule completion within 5 days
3. Complete sections in order

## Philosophy

The goal is not just to “get through” a show —
it is to systemise quality, reduce friction, and improve week-on-week.

3–4 hours total (spread across week)
