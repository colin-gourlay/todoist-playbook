# Radio Show Week

Everything needed to run a full radio show week, from pre-production and playlists through to upload, comms, and social media.

This bundle collects the complete end-to-end workflow for producing and broadcasting a radio show.

---

## Included Templates

| Template | Purpose |
|----------|---------|
| [Radio Show System](../../templates/radio-show-system/) | Full pre-production, live broadcast, post-production, and reflection workflow |

---

## When to Use This Bundle

- At the start of each radio show week
- As a repeatable weekly production system
- When onboarding someone new to the show workflow

---

## Suggested Setup

1. Import **Radio Show System** at the start of each show week
2. Rename the project to reflect the show date or episode (e.g. `Show – 15 Mar`)
3. Work through the sections in order: Pre-Production → Pre-Live Logistics → Studio Setup → Live Broadcast → Post-Production → Reflection

---

## How to Import

1. Navigate to the template folder linked above
2. Download the `template.csv` file
3. In Todoist: create a new project → Import from CSV
4. Rename the project as suggested in the template README
