# Radio Show Week Kit

A complete operational system for producing a weekly radio show — from creative preparation through to publishing, promotion, and reflection.

This bundle combines focused templates into a reusable weekly workflow that covers every stage of the radio production process.

---

## Included Templates

| Template | Purpose | Required |
|----------|---------|----------|
| [Radio Show Core](../../csv-templates/radio-show-core/) | Core weekly workflow — creative prep, logistics, studio setup, and live broadcast | ✅ Every week |
| [Radio Show Promotion](../../csv-templates/radio-show-promotion/) | Pre-show and post-show promotion across social channels | ✅ Every week |
| [Radio Show Post Production](../../csv-templates/radio-show-post-production/) | Publishing, media distribution, archiving, and reflection | ✅ Every week |
| [Radio Show Guest Feature](../../csv-templates/radio-show-guest-feature/) | Guest or feature segment preparation and follow-up | ⚙️ Optional |
| [Artist Interview Invite Workflow](../../csv-templates/artist-interview-invite-workflow/) | Artist or band outreach workflow with copy/paste invite and follow-up email templates | ⚙️ Optional |

---

## When to Use This Bundle

- At the start of each radio show week
- As a repeatable end-to-end production system
- When onboarding a new presenter, producer, or show assistant
- When running a special edition or themed show week

---

## When to Use the Optional Template

Use **Radio Show Guest Feature** when the show includes:

- A live or pre-recorded interview
- An artist spotlight or in-depth feature
- Festival coverage or a special edition
- A recurring guest contributor

Use **Artist Interview Invite Workflow** when you are actively sourcing and inviting new guests for upcoming shows.

Skip optional templates in standard weeks with no guest segment.

---

## Suggested Setup

1. At the start of each show week, import the three required templates
2. If the week includes a guest or feature segment, also import **Radio Show Guest Feature**
3. If you are in outreach mode for future episodes, import **Artist Interview Invite Workflow**
4. Rename each project using your show name and a descriptor — for example, for a show called *Sundown Sessions*:

   | Project | Suggested Name |
   |---------|----------------|
   | Radio Show Core | `Sundown Sessions Core – Week of [date]` |
   | Radio Show Promotion | `Sundown Sessions Promotion – Week of [date]` |
   | Radio Show Post Production | `Sundown Sessions Post Production – [date]` |
   | Radio Show Guest Feature | `Sundown Sessions – [Guest Name] Feature` |
   | Artist Interview Invite Workflow | `Sundown Sessions Guest Invite – [Artist Name]` |

5. Work through the templates in this order:
   - **Radio Show Core** — runs throughout the week, culminating in the live broadcast
   - **Artist Interview Invite Workflow** *(if applicable)* — complete during guest booking cycles
   - **Radio Show Guest Feature** *(if applicable)* — complete before broadcast day
   - **Radio Show Promotion** — pre-show tasks before broadcast; post-show tasks after
   - **Radio Show Post Production** — complete on broadcast day or the day after

---

## How to Import

1. Navigate to each template folder linked in the table above
2. Download the `template.csv` file from each folder
3. In Todoist: create a new project → Import from CSV
4. Rename each project as suggested above
5. Work through tasks in the recommended order
