# Radio Show Week Kit

A complete operational system for producing a weekly radio show — from creative preparation through to publishing, promotion, and reflection.

This bundle combines four focused templates into a reusable weekly workflow that covers every stage of the radio production process.

---

## Included Templates

| Template | Purpose | Required |
|----------|---------|----------|
| [Radio Show Core](../../templates/radio-show-core/) | Core weekly workflow — creative prep, logistics, studio setup, and live broadcast | ✅ Every week |
| [Radio Show Promotion](../../templates/radio-show-promotion/) | Pre-show and post-show promotion across social channels | ✅ Every week |
| [Radio Show Post Production](../../templates/radio-show-post-production/) | Publishing, media distribution, archiving, and reflection | ✅ Every week |
| [Radio Show Guest Feature](../../templates/radio-show-guest-feature/) | Guest or feature segment preparation and follow-up | ⚙️ Optional |

---

## When to Use This Bundle

- At the start of each radio show week
- As a repeatable end-to-end production system
- When onboarding a new presenter, producer, or show assistant
- When running a special edition or themed show week

---

## When to Use the Optional Template

Use **Radio Show Guest Feature** when the show includes:

- A live or pre-recorded interview
- An artist spotlight or in-depth feature
- Festival coverage or a special edition
- A recurring guest contributor

Skip this template in standard weeks with no guest segment.

---

## Suggested Setup

1. At the start of each show week, import the three core templates
2. If the week includes a guest or feature segment, also import **Radio Show Guest Feature**
3. Rename each project to reflect the show date:

```
Sundown Sessions – Week of [date]
```

4. Work through the templates in this order:
   - **Radio Show Core** — runs throughout the week, culminating in the live broadcast
   - **Radio Show Guest Feature** *(if applicable)* — complete before broadcast day
   - **Radio Show Promotion** — pre-show tasks before broadcast; post-show tasks after
   - **Radio Show Post Production** — complete on broadcast day or the day after

---

## How to Import

1. Navigate to each template folder linked in the table above
2. Download the `template.csv` file from each folder
3. In Todoist: create a new project → Import from CSV
4. Rename each project as suggested above
5. Work through tasks in the recommended order
