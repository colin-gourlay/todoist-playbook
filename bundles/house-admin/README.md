# House Admin Starter Kit

A curated set of templates for keeping on top of household administration — bills, renewals, vehicle maintenance, and property upkeep.

---

## Included Templates

| Template | Purpose |
|----------|---------|
| [House Admin](../../csv-templates/house-admin/) | Annual household checklist — bills, insurance renewals, MOT, property maintenance, and financial records |

---

## When to Use This Bundle

- At the start of a new year as a full household audit
- When approaching insurance or contract renewal dates
- When a vehicle MOT or service is coming up
- Whenever you feel household admin has slipped

---

## Suggested Setup

1. Import **House Admin** twice per year (January and July work well)
2. Rename the project to: `House Admin – [Year]`
3. Work through sections at your own pace — prioritise items with the nearest deadlines

---

## How to Import

1. Navigate to the template folder linked above
2. Download the `template.csv` file
3. In Todoist: create a new project → Import from CSV
4. Rename the project as suggested in the template README
